Name collisions worth knowing about

Two components called "CTABanner" are not the same file

@/components/core/home/CTABanner (Home only) and @/components/common/Industry/CTABanner (all 6 Solutions, all 6 Industries, both Services pages) share a name but are unrelated components. Editing one to fix a Home bug won't touch the other 14 pages, and vice versa.

SolutionHero and ServiceHero look related but aren't interchangeable

All 6 Solutions pages use SolutionHero; /services/port-services uses a separate ServiceHero from the same @/components/common/Service/ folder. /services/port-network uses neither — it uses IndustryHero instead.
